# Birdsvew — Landing Page & Waitlist

This repository contains the pre-launch landing page for Birdsvew, a platform that generates AI-powered visual STEM lessons. 
Ask a STEM question, and Birdsvew turns it into a fully rendered, narrated visual lesson — built for understanding, not distraction.

The site collects early-access signups and lets visitors watch short, animated preview lessons in the style of explainer videos.

🔗 **Live site:** [birdsvew.com](https://birdsvew.com)

---

## What this is

This is a single-page marketing and waitlist site — the front door for Birdsvew before the full product launches.
